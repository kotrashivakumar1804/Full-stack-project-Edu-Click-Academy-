# 🎓 EduClick Academy

A full-stack Educational Management System built using Flask, Python, HTML, CSS, JavaScript, and AI-assisted development tools.

## 🚀 Project Overview

EduClick Academy is a web-based platform designed to simplify student enrollment, course management, announcements, and administrative operations through a user-friendly dashboard.

This project demonstrates my ability to combine software development skills with modern AI tools to build real-world applications efficiently.

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/kotrashivakumar1804/Full-stack-project-Edu-Click-Academy-.git
```

### Navigate to Project

```bash
cd Full-stack-project-Edu-Click-Academy-
```

### Create Virtual Environment

```bash
python -m venv venv
```

### Activate Virtual Environment

Windows:

```bash
venv\Scripts\activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Application

```bash
python app.py
```
